# Правила

## Вступ

**Базова мова – українська.**

## Базові правила

### Закінчення слів:

1. **Якщо** слово закінчується на _приголосну літеру_, ми додаємо < ꙏ > до кінця слова.
2. **Якщо** слово закінчується на < й >, ми замінюємо цю літеру на < јъ >.
3. **Якщо** слово закінчується на < ль >, ми замінюємо цю літеру на < ԉ >.
4. **Якщо** слово закінчується на < нь >, ми замінюємо цю літеру на < ԋ >.
5. **Якщо** слово закінчується на < сь >, ми замінюємо цю літеру на < ԍ >.
6. **Якщо** слово закінчується на < ть >, ми замінюємо цю літеру на < ԏ >.
7. **Якщо** слово закінчується на < зь >, ми замінюємо цю літеру на < ԅ >.
8. **Якщо** слово закінчується на < дь >, ми замінюємо цю літеру на < ԃ >.
9. **Якщо** слово закінчується на < ць >, ми замінюємо цю літеру на < წ >.
10. **Якщо** слово закінчується на < чь >, ми замінюємо цю літеру на < ჭ >.

### Діакритичні знаки:

- #### Над приголосними:

  1. Тильда ( ~ ) над приголосною літерою означає, що вона подвоюється.
  2. Гачек ( ˇ ) над приголосною означає що перед нею іде буква < з >.

  Приклад:

  - в̌ читається як < зв >
  - в̄ читається як < вв >

- #### Над голосним:

  1. Макрон ( ˉ ) над голосною означає що вона подовжується
  2. Умлаут ( ¨ ) означає що буква м'яка. Застосовується до: < ә >, < ъ >. В <а> <е> <и> <о> <у> <я> <ю> <є> означає що після них іде < й >
  3. Умлаут і Макрон ( ¨̄ ) означає що буква подовження і друга буква пом'якшена.
  4. Бреве ( ˘ ) над < о > і < у > означає що після них іде < в >.
  5. Кружок ( ˚ ) над < а > і < у > означає що після них іде < о >.

  Приклади:

  - ӓ̄ читається як < ая >
  - а̄ читається як < аа >
  - ӛ читається як < я >
  - ў читається як < ув >
  - а̊ читається як < ао >
